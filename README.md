# API VENDAS .NET

## Sobre o Projeto
API construída com ASP.NET Core como parte de estudos em desenvolvimento backend.

## Tecnologias
- ASP.NET Core
- Entity Framework Core
- C#

## Como Rodar
1. Clone o repositório
2. Abra no Visual Studio / VS Code
3. Configure o `appsettings.json`
4. Execute a API

## Endpoints
- GET /produtos
- POST /clientes
